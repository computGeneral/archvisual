ARCH VISUALIZER

This is a data visualizing tool set for architecture exploration, including:

1. MetricViewer: used for performance counter/metrics visualization on demand

2. TraceViewer: Google Catapult trace viewer for timeline visualization

3. CrossViewer: combined MetricViewer and TraceViewer together

4. PerfettoViewer: Google Perfetto trace viewer with offline self-hosted UI and server file browser
